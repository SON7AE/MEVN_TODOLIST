## 1.개발환경 세팅

### - 백엔드 : Node.js로 구성 (Express.js)

`npm install express`

### - DataBase : MongoDB로 구성

`npm install mongoose`

### - DB ID 및 PASSWORD 보안

`npm install dotenv`

### - Nodemon 설치

`npm install -g nodemon`
